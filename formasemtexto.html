Shape-oustslide

essa propriedade permite que voce defina uma forma para que o texto não se envolva e flua ao seu redor

Junto a estra propriedade temos algumas formas básicas

o elemento que vamos aplicar a forma com a propriedade Shape-oustslide

precisa ter a propriedade float

  • inset()
  • circle()
  • ellipse()
  • polygon()

Insert()

pode ser usado para criar um retangulo/quadrado com uma deslocamento opcional para o texto que o envolver
